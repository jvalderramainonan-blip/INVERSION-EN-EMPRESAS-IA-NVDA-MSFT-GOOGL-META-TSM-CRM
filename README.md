En este apartado se analiza econometricamente utilizando la econometria financiera y modelos de regimines cambiantes de Markov Shitiching para estimar los valores VaR de una inversión y las probabilidades de mantenerse  en recesión, expansión y auge. Todo esto se realiza con la finalidad de evaluar un portafolio óptimo de 6 empresas de IA.
